Agent，就是套在大模型外面的一个 for 循环。

大模型本身是无状态的：每次调用都是独立的一锤子买卖，模型并不"记得"上一次说过什么。 所谓"记忆"，全是工程拼出来的——每一轮，我们都把到目前为止全部历史加上最近一次工具结果 重新塞给模型，让它决定"再调一次工具"还是"给最终答案"。

下面每张卡片就是一次 LLM 调用：我们送进去了什么、它返回了什么、（如果它要调工具）工具又回了什么。 从第二轮起，输入里★ 黄色高亮的就是本轮新增进来的内容—— 这正是 agent "记住"上一轮工具结果的方式。
