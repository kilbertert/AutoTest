本 skill 用于指导生成有辨识度、生产级可用的前端界面，避开俗套的"AI slop"美学。请直接实现真实可运行的代码，并在审美细节与创意取舍上倾注极致的关注。

用户会给出前端需求：一个组件、页面、应用或界面。他们可能附带关于用途、受众或技术约束的背景。

## 设计思维

下笔前，先理解语境并锁定一个**大胆**的美学方向：

- **目的**：这个界面解决什么问题？谁来用？
- **基调**：挑一个极致方向——极简克制 / 极繁堆叠 / 复古未来 / 自然有机 / 奢华精致 / 玩具趣味 / 编辑杂志风 / 粗野主义 / Art Deco 几何 / 柔和马卡龙 / 工业实用，等等。把这些当灵感，再设计出真正贴合该方向的味道。
- **约束**：技术要求（框架、性能、可访问性）。
- **差异化**：什么让它令人难忘？用户会记住的那"一个东西"是什么？

**关键**：选定一个清晰的概念方向并精准执行。极繁与极简都能成立——成败在于"刻意为之"，而非强度。

随后实现满足以下标准的可运行代码（HTML/CSS/JS、React、Vue 等）：

- 生产级、功能完整
- 视觉冲击力强、令人难忘
- 整体连贯，有清晰的设计观点
- 每个细节都经过精心打磨

## 前端美学准则

重点关注：

- **字体**：选漂亮、独特、有趣的字体。避免 Arial、Inter 这类通用字体；转而选择能提升美感的、出人意料且有性格的字体。用一个有辨识度的展示字体（display）搭配一个耐看的正文字体。
- **色彩与主题**：确立统一连贯的美学，用 CSS 变量保持一致。"主色占大面积 + 少量利落的强调色"远胜于怯生生、平均分布的配色。
- **动效**：用动画做点睛与微交互。HTML 优先纯 CSS 方案；React 在可用时优先 Motion 库。把力气集中在高价值时刻——一次精心编排、带错峰渐入（animation-delay）的页面加载，比满屏零散的微交互更有惊喜感。善用滚动触发与出人意料的 hover 状态。
- **空间构成**：不走寻常布局。不对称、重叠、对角线流动、打破网格的元素。要么大胆留白，要么有控制地堆密度。
- **背景与视觉细节**：营造氛围与层次，而非默认铺纯色。叠加与整体基调相符的效果与纹理：渐变网格、噪点纹理、几何图案、半透明分层、戏剧化阴影、装饰性边框、自定义光标、颗粒叠层等。

**绝不**使用通用的 AI 生成美学：烂大街字体（Inter、Roboto、Arial、系统字）、俗套配色（尤其是白底紫色渐变）、可预测的布局与组件套路、缺乏场景个性的模板化设计。

请创造性地理解需求，做出真正为该语境设计的、出人意料的选择。没有两个设计应该长一样。在浅色与深色主题、不同字体、不同美学之间主动变化。**绝不**每次都收敛到常见选择（例如 Space Grotesk）。

**重要**：让实现的复杂度匹配美学愿景。极繁设计需要繁复的代码、大量动画与效果；极简或精致设计则需要克制、精准，以及对间距、字体、微妙细节的细致关注。优雅源于把愿景执行到位。

记住：你有能力做出非凡的创意作品。不要保守——当你跳出框架、完全投入一个独特愿景时，展示真正能创造出的东西。
